# Portafolio de Sergio Antunez Rios

Portafolio bilingüe sobre IA aplicada, producto empresarial e Ingeniería Electromédica.

## Desarrollo local

```bash
npm install
npm run dev
```

## Verificación

```bash
npm run lint
npx tsc --noEmit
npm run build
```

Sitio público: [portafolio-henna-five.vercel.app](https://portafolio-henna-five.vercel.app/)

La evidencia empresarial publicada está agregada y omite deliberadamente arquitectura, proveedores y detalles internos.
